> UX를 기반으로 UI 구현
> UI를 바탕으로 웹/앱 서비스 구축

서비스를 구축하는데 "왜 필요한가?" 라는 질문에 답을 얻고
해당 질문에 맞는 화면을 구현하고 이를 서비스로 실행

* 서비스 개발 시 

- 정보 수집 : 뉴스 / 기업 데이터 / 정부 자료
- 실제 실무 : 카드 / 스마트폰 데이터 사용량

- 필드 리서치 : 현업 관계자 / 산업 담당자 리서치
- 데스크 리서치 : 공공 데이터 포털 / 트랜드 랩 / 연관 키워드 사이트

예 1)
1. 문제인식
> 최근 차를 선호하는 사람 증가

2. 정보 수집
> 네이버 데이터 랩 / 광고 추이 분석

3. 타겟
> 30-40 여성

4. 페르소나
> 다이어트에 관심이 많은 사람들

5. 목표

6. 서비스 도출


예 2)


JS > Class

예) 교보문고
> 매일 신규 등록되는 책
> 객체 자료 구조
> 저자 / 가격 / 페이지 / 출간일 / 출판사명 등

Class

저자 / 가격 / 페이지 / 출간일 / 출판사명 등 세부 정보만 교체 가능
