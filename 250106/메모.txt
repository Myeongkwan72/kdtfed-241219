"라이브러리 & 프레임워크"
- 기존 문법을 쉽고 빠르게 사용할 수 있도록 하는 요소들

라이브러리 : 기존에 존재하는 언어의 문법 체계를 토대로
                그 위에 본인들의 커스텀 문법을 얹힌 상태

프레임워크 : 완벽하게 본인들만의 문법 체계를 구축하여
                본인들의 시스템을 개발자들이 별도로 학습

- CSS 속성
> 단위
> 색상
> 가시속성

- display

block
- 특정 요소가 보여지도록 하는 속성

inline
- 특정 요소가 보여지도록 하는 속성

inline-block
- inline-block 속성 안의 요소들이 하나의 행에 포함되는 경우
  inline처럼 공통적으로 사용
- inline-block 속성 안의 요소들이 하나의 행을 벗어나는 경우
  block처럼 독립적으로 사용

inline과 inline-block의 차이
1) 너비, 높이 값이 적용되는가
2) 마진, 패딩 값이 어디까지 적용되는가


none
- 특정 요소가 보이지 않도록 하는 속성
- 공간을 없애는 속성

visiblity
- 공간을 유지한 채로 보이지 않게 하는 속성

margin
- 특정 요소 바깥 여백을 얼마나 부여할 것인가

padding
- 특정 요소 내부 여백을 얼마나 부여할 것인가
